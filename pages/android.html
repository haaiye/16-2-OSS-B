안드로이드의 전체적 구조

안드로이드는 왼쪽 그림과 같은 구성 요소로 구성되어 있으며 응용 프로그램, 응용 프로그램 프레임워크, 라이브러리, 안드로이드 런타임, 리눅스 커널의 총 5개의 계층으로 분류되어 있다.

1. APPLICATIONS : 구글에서 Java로 자체 제작하였다. Application에는 연락처, 전화 브라우저 등 우리가 일상생활에서 사용하는 휴대폰 앱이 굉장히 많다.

2. APPLICATION FRAMEWORK : 구글에서 Java로 자체제작하였다. 주로 활동매니저, 윈도우매니저 등으로 구성되어 있다. 말 그대로 Application을 돌리기 위한 Framwork이다.

3. LIBRARIES : C, C++ 언어를 기반으로 이루어져 있으며 안드로이드 앱개발에 필요한 오픈소스 기반의 솔루션들이 모여있다. 하지만 취약점이 많아서 보안상 문제점이 Library 쪽에서 많이 발생한다.

4. ANDROID RUNTIME : Application 제작은 JVM으로 개발 후 최종적으로 Dalvik 코드로 변경해서 Target 장치에서 구동한다. 안드로이드는 리눅스, 라이브러리가 C, C++로 개발되어지고 이루어졌지만 자바로 개발이 가능한 이유가 JVM이라는 자바 가상머신이 얹어졌기 때문에 자바로 코딩이 가능하다. 하지만 MySQL, ORACLE 같은 엔진들은 너무 무겁기 때문에 안드로이드에서는 사용하기가 어렵다.

안드로이드 5.0부터는 Dalvik가상머신이 아닌 ART가상머신으로 바뀌었다. 그 이유는 application을 실행할 때 무조건 가상머신을 통해 실행시켜야 하니 배터리 효율이나 반응속도가 떨어졌기 때문이다. 그래서 구글은 Dalvik을 대체할 ART모드를 내놓았는데 이 것은 Dalvik처럼 가상머신을 통하지 않고 바로 실행한다. 그 덕분에 Dalvik의 단점이었다 앱실행속도, 배터리효율이 증가하게 되었다. 그러나 앱설치시 기존 Dalvik에 비해 용량이 증가했고 설치시간이 더 오래걸린다는 단점이 있다.

5. LINUX KERNEL : C, C++언어를 기반으로 이루어져 있으며 리눅스 계열로 이루어져 보안이 강력하다. 주로 드라이버로 이루어져 있으며 화면드라이버, 키보드 드라이버, 카메라 드라이버 등으로 구성되어 있다.

안드로이드 운영 체제 버전

2007년 11월 안드로이드 1.0 버전부터 시작한다. 안드로이드는 구글과 오픈 핸드셋 얼라이언스가 개발한 모바일 운영 체제로, 최초 배포판 이후로 꾸준히 업데이트되었다. 업데이트는 일반적으로 버그 수정과 새로운 기능 추가 등이다. 2009년 4월부터 각각의 안드로이드 버전은 디저트 이름을 바탕으로 코드네임을 붙여 개발되고 있다. 각 버전들은 알파벳 C 부터 알파벳 순으로 명명되었으며 A와 B의 공식 명칭은 존재하지 않는다.

안드로이드 1.0 애플파이(1.0 API Level 1) - 안드로이드 마켓/웹 브라우저/카메라 지원 - 하지만 이 버전은 해상도, 화이트 밸런스, 품질 및 기타 설정 옵션이 부족했다./G메일/구글 주소록과 동기화가 가능한 피플 앱/구글 지도 (위치 찾기, 스트리트 뷰 포함)/구글 싱크/구글 검색/구글 토크/SMS, MMS/유튜브/Wi-Fi, 블루투스 지원/기타 앱 : 알람시계, 계산기, 전화, 홈 스크린 (런처), 사진 (갤러리), 설정

안드로이드 6.0 마시멜로

안드로이드 7.0 누가 - 안드로이드 누가는 차세대 안드로이드 버전 7.0의 공식명칭이다. 누가라는 이름이 확정되기 전에는 나초, 누텔라 같은 알파벳 N 으로 시작하는 간식이름들이 많이 떠돌아 다녔으나 악마의 잼 '누텔라(nutella)' 라는 주장이 가장 강했으나 사람들의 예상과는 다르게 '누가'라고 확정되었다. 안드로이드 7.0 누가는 구글 프리뷰가 처음으로 나오고, 미국 현지시간으로 2016년 8월 22일, 국내시간 2016년 8월 23일에 출시됐다. 현재 넥서스 5X, 넥서스 6P, 넥서스 6 등과 함께 주력기기들은 OTA 업데이트가 보내지고 있다. 정식 이미지도 나왔다고 한다. LG전자는 LG 브릿지 업그레이드 프로그램으로 LG G5 LG U+ 통신사용으로 누가 프리뷰를 국내시간 2016년 8월 31일에 배포 하였다. LG는 9월 29일 나올 예정인 LG V20으로 첫 상용화 되었다

안드로이드 스튜디오

Android Studio는 안드로이드를 위한 통합 개발 환경 (IDE)이다. 2013년 5월 16일, 구글 I/O 컨퍼런스에서 구글의 제품 관리자인 Ellie Powers에 의해서 발표되었다. 아파치 라이선스 2.0으로 배포된다. 기존에는 이클립스를 많이 사용했지만 최근애는 intelliJ인 안드로이드 스튜디오로 많이 개발하는 추세이다.
